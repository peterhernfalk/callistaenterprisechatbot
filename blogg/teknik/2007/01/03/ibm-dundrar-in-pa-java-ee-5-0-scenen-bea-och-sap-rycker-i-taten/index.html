IBM dundrar in på Java EE 5.0 scenen! BEA och SAP rycker i täten!!!

03 January 2007 // Ola Deibitsch

Efter att IBM nu har släppt en alphaversion av EJB 3.0 som ett “feature pack” till WebSphere 6.1, så har man därmed tagit det första steget mot att stödja Java EE 5.0! Ferdinand har alltså äntrat scenen för att stångas med dem andra tjurarna i kampen mot en fullständing Java EE 5.0 implementation. IBM har dock en del att hämta in då BEA är en bra bit före med en “technology preview” av hela Java EE 5.0 implementation och inte tala om SAP som redan fått in alla getterna i hagen! En annan leverantör som ligger och simmar i kölvattnet på både BEA och SAP är Geronimo, vilken slutet av december förra året släppte den första milstolpen av deras implementation av Java EE 5.0, dvs 2.0-M1.
