Håkan Dahl

Arkitekt, senior utvecklare

Håkan har en bred erfarenhet av systemutveckling i ett flertal brancher. Det handlar ofta om tekniskt ledande roller, med inslag av såväl arkitektur som praktisk utveckling. Inte sällan handlar det även om att stötta ett team med både teknisk expertis och utbildningsinsatser. Allra bäst trivs Håkan i projekt med inslag av teknik och komplexa utmaningar.

Varför Callista?

Inspirerande miljö där väldigt kunniga och samtidigt prestigelösa kollegor delar med sig av kunskap. Stort eget ansvar med möjligheter att utvecklas inom de områden som intresserar mig. Viljan inom företaget att alltid lära lite mer har heller inte mattats under årens lopp.

Java 9 Modulsystem (Jigsaw) Presentation

27 January 2016 // Håkan Dahl

Presentation från Cadec 2016

Det nya modulssystemet (projekt Jigsaw) är huvudnumret på den rätt omfattande feature-listan för Java 9. Ursprungligen tänkt att ingå i Java 7 har modulsystemet nu tagit form. Modulsystemet ska råda bot på problem relaterade till classpath-röra, svag inkapsling i Java och ett monolitiskt JDK.
